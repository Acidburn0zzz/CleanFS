CleanFS
=======

A proposal for a new [Flyspray](http://flyspray.org/) theme.

Usage
-----

- In Flyspray 0.9.9.6:
  - put files in your `/flyspray/themes/` directory
  - overwrite your `/flyspray/templates/` with the new ones (you might want to backup them first)
